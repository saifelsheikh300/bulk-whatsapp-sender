# واتساب الطيار 📲

تطبيق Android لإرسال رسائل واتساب جماعية لعدد كبير من جهات الاتصال بشكل تلقائي، مع دعم الرسائل المتعددة العشوائية والملفات (صور، فيديو، صوت) لتقليل احتمالية الحظر.

---

## ✨ الميزات

### 1. الإرسال الجماعي التلقائي
- اختيار عدد غير محدود من جهات الاتصال من قائمة جهات الهاتف
- إرسال تلقائي متتابع لكل جهة اتصال بدون تدخل يدوي
- يعمل عبر **Accessibility Service** يتحكم في واتساب نيابة عن المستخدم (فتح المحادثة، كتابة النص، الضغط على إرسال)

### 2. دعم واتساب العادي وواتساب بيزنس
- اختيار أي نسخة تُستخدم للإرسال (`com.whatsapp` أو `com.whatsapp.w4b`) من شاشة الإعدادات الرئيسية

### 3. رسائل متعددة عشوائية (Message Variants / Anti-Ban)
- إمكانية إضافة عدد غير محدود من نسخ الرسائل (مش محدد بعدد معين)
- عند الإرسال، يتم اختيار رسالة عشوائية لكل جهة اتصال من بين النسخ المضافة
- الهدف: تجنّب الحظر، لأن واتساب يكتشف الأنماط المتكررة (نفس النص لمئات الجهات)؛ كل رسالة مختلفة عن التانية يقلل هذا النمط

### 4. إرفاق ملفات (صور / فيديو / صوت / أي ملف)
- كل رسالة (variant) ممكن تحتوي على نص فقط، أو ملف فقط، أو الاثنين مع بعض (الملف + نص كـ caption)
- اختيار الملف من معرض الهاتف بالكامل عبر منتقي ملفات النظام (`*/*`)
- الإرسال يتم مباشرة لجهة الاتصال عبر `ACTION_SEND` مع تحديد الشخص بالـ `jid` (دون فتح شاشة اختيار جهة اتصال يدوية)

### 5. تأخير قابل للتعديل بين كل رسالة (Anti-Ban)
- شريط تمرير (Slider) لاختيار التأخير من 5 إلى 30 ثانية بين كل رسالة وأخرى
- تحذير مرئي عند اختيار أقل من 7 ثواني (خطر حظر الحساب)

### 6. شاشة تقدّم الإرسال (Progress Screen)
- عرض حي لعدد الرسائل المرسلة والمتبقية ونسبة الإنجاز
- عرض اسم جهة الاتصال الجاري إرسال الرسالة لها حالياً
- عدّاد تنازلي للوقت المتبقي قبل إرسال الرسالة التالية

### 7. إيقاف مؤقت / استئناف / إيقاف كامل
- إمكانية إيقاف عملية الإرسال مؤقتاً في أي وقت ثم استئنافها من حيث توقفت
- إمكانية إنهاء الجلسة بالكامل
- منع الإرسال المكرر لنفس جهة الاتصال إذا تم إرسال رسالتها بالفعل (Double-send Prevention)

### 8. فقاعة عائمة (Floating Bubble)
- فقاعة تظهر فوق باقي التطبيقات أثناء عملية الإرسال لمتابعة الحالة دون الحاجة للعودة للتطبيق

### 9. واجهة عربية بالكامل (RTL)
- تصميم كامل من اليمين لليسار
- خط Cairo العربي في كل أنحاء التطبيق
- ألوان مستوحاة من هوية واتساب (أخضر / خلفية داكنة)

### 10. تطبيع أرقام الهواتف المصرية تلقائياً
- يقبل الرقم بأي صيغة ويحوّله تلقائياً للصيغة الدولية الصحيحة:
  - `01012345678` ← `+201012345678`
  - `1012345678` ← `+201012345678`
  - `00201012345678` ← `+201012345678`
  - `+201012345678` ← بدون تغيير

## 🏗️ البنية التقنية (Architecture)

```
app/src/main/java/com/seif/bulkwhatsapp/
├── data/
│   ├── Contact.kt                 # موديل جهة الاتصال وحالة الإرسال
│   ├── ContactsRepository.kt      # جلب جهات الاتصال من الهاتف
│   └── SendSession.kt             # موديل الجلسة + الرسائل المتعددة (MessageVariant) + SessionManager
├── service/
│   ├── WhatsAppAccessibilityService.kt  # المحرك الأساسي: يتحكم في واتساب عبر Accessibility API
│   └── FloatingBubbleService.kt         # خدمة الفقاعة العائمة
├── ui/
│   ├── MainActivity.kt            # الشاشة الرئيسية (اختيار الجهات، الرسائل، التأخير، البدء)
│   ├── ContactsActivity.kt        # شاشة اختيار جهات الاتصال
│   ├── ContactsAdapter.kt         # عرض قائمة جهات الاتصال (RecyclerView)
│   └── ProgressActivity.kt        # شاشة متابعة تقدّم الإرسال
└── utils/
    └── PhoneUtils.kt              # تطبيع أرقام الهواتف المصرية
```

### كيف يعمل الإرسال؟

1. المستخدم يختار جهات الاتصال + يكتب رسالة واحدة أو أكثر (نص و/أو ملف) + يحدد التأخير
2. عند الضغط على "ابدأ الإرسال"، يتم إنشاء `SendSession` وتفعيل `WhatsAppAccessibilityService`
3. لكل جهة اتصال:
   - يتم اختيار رسالة عشوائية (`pickVariant()`) من القائمة
   - **نص فقط:** يُفتح واتساب على شات الشخص مباشرة عبر `https://api.whatsapp.com/send?phone=...`، ثم يكتب الـ Accessibility Service النص في حقل الإدخال (`entry`) ويضغط زر الإرسال (`send`)
   - **ميديا:** يُرسل `Intent.ACTION_SEND` مباشرة مع تحديد الشخص عبر `jid` (`<phone>@s.whatsapp.net`)، ثم يضغط الـ Accessibility Service على زر الإرسال في شاشة معاينة الملف
   - بعد الإرسال بنجاح، ينتظر التطبيق المدة المحددة (Delay) قبل الانتقال للجهة التالية

## 🔐 الصلاحيات المطلوبة

| الصلاحية | الغرض |
|---|---|
| `READ_CONTACTS` | قراءة جهات الاتصال لاختيار المستلمين |
| `BIND_ACCESSIBILITY_SERVICE` | التحكم في واجهة واتساب لإرسال الرسائل تلقائياً |
| `SYSTEM_ALERT_WINDOW` | عرض الفقاعة العائمة فوق التطبيقات الأخرى |
| `READ_MEDIA_IMAGES` / `READ_MEDIA_VIDEO` / `READ_MEDIA_AUDIO` | الوصول للملفات المرفقة (صور/فيديو/صوت) |
| `FOREGROUND_SERVICE` | تشغيل خدمة الفقاعة في الخلفية بثبات |

## ⚠️ تنبيهات هامة

- **معدّل الإرسال:** التأخير الموصى به 10 ثانية أو أكثر بين كل رسالة لتقليل احتمالية حظر الحساب من واتساب
- **الرسائل المتعددة:** كلما زاد عدد نسخ الرسالة المختلفة، قلّ احتمال اكتشاف واتساب لنمط الإرسال الجماعي
- التطبيق يعتمد على بنية واجهة واتساب الداخلية (`view IDs`)، وأي تحديث كبير من واتساب قد يتطلب تحديث مماثل في التطبيق
