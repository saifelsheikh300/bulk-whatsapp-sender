# ✈️ وتساب الطيار

تطبيق أندرويد لإرسال رسائل واتساب جماعية بشكل ذكي وآمن.

## المميزات
- 📱 اختيار جهات الاتصال من التليفون
- ✅ دعم واتساب العادي والبيزنس
- ⏱️ تأخير 10 ثواني بين كل رسالة لتجنب الحظر
- 🔍 بحث في جهات الاتصال
- 📊 شاشة تقدم مباشر
- 🎨 واجهة داكنة احترافية
- ⚡ خفيف جداً على الأجهزة (1-4 RAM)

## طريقة الاستخدام
1. فعّل خدمة الإمكانية من الإعدادات
2. اختر واتساب عادي أو بيزنس
3. حدد جهات الاتصال
4. اكتب الرسالة
5. اضغط "ابدأ الإرسال"

## متطلبات
- Android 5.0+
- واتساب أو واتساب بيزنس مثبت
